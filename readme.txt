This directory represents the data analysis aspect of team 4s submission for the
2017 research practicum.

Raw data needed to run these scripts is accessible in the following locations.

Raw transit data: https://data.dublinked.ie/dataset/dublin-bus-gps-sample-data-from-dublin-city-council-insight-project
Raw weather data: http://www.met.ie/climate-request/


In order to clean data and create prediction models, please run these notebooks in the following
order:

data separating 2
data cleaning 2
clean by distance
adding weather data (all)
random forest regression (all lines) 3

When running these scripts, please change the path names of input and output directories to
reflect your filesystem.
